# Optimal BST Visualizer

This project visualizes the construction of an Optimal Binary Search Tree (OBST) using Dynamic Programming.

## Features
- Generate Optimal BST from keys and frequencies
- Compare with Normal BST
- Tree Visualization
- Search Simulation with animation
- Dynamic Programming Table
- Step-by-step OBST construction
- Performance Graph

## Technologies Used
- Frontend: React.js
- Backend: Node.js (Express)
- Algorithm: C++
- Visualization: react-d3-tree, Chart.js

## How to Run

### Backend
cd backend
node server.js

### Frontend
cd frontend
npm start
